## 📌 Overview

An end-to-end **automated visual inspection system** for detecting defects in textile fabrics using classical computer vision and machine learning. The system applies a bank of **Gabor filters** at multiple orientations and frequencies to extract rich texture features from fabric images, then uses a **One-Class SVM** and a **Dense Autoencoder** trained only on defect-free samples to flag anomalous regions — no defect labels required during training.

> ✅ Runs fully on **Google Colab** with zero setup — includes a built-in synthetic dataset generator.

```
│                     INPUT PIPELINE                          │
│  Camera / Scanned fabric roll → Grayscale image             │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐

│  1. Resize to 256×256                                       │

│  3. Gaussian Blur (3×3)           — suppress sensor noise   │

                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                   GABOR FILTER BANK                         │
│  5 frequencies  ×  8 orientations  =  40 filtered maps      │
│  Parameters: ksize=31, sigma=4.0, gamma=0.5                 │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                  PATCH FEATURE EXTRACTION                   │
│  Sliding window: patch=32×32, stride=16                     │
│  Per patch per filter: mean · std · energy · entropy        │
│  Feature vector size: 40 filters × 4 stats = 160 dims       │
└───────────────────────────┬─────────────────────────────────┘
                            │
               ┌────────────┴────────────┐
               ▼                         ▼
┌──────────────────────┐   ┌─────────────────────────────────┐
│   ONE-CLASS SVM      │   │        AUTOENCODER              │
│  kernel=rbf, nu=0.1  │   │  Encoder: 160→128→64→16        │
│  Trained on GOOD     │   │  Decoder: 16→64→128→160        │
│  patches only        │   │  Anomaly = high recon. error    │
└──────────┬───────────┘   └──────────────┬──────────────────┘
           │                              │
           └──────────────┬───────────────┘
                          ▼
┌─────────────────────────────────────────────────────────────┐
│              DEFECT LOCALISATION & OUTPUT                   │
│  Anomaly heatmap → Otsu threshold → Morphological cleanup   │
│  → Bounding boxes → PASS / FAIL verdict                     │
└─────────────────────────────────────────────────────────────┘
```

## 🔬 How Gabor Filters Work

A Gabor filter is a sinusoidal wave modulated by a Gaussian envelope — designed to detect texture at a specific **orientation** and **spatial frequency**, similar to how the human visual cortex processes texture.

```
G(x, y; λ, θ, σ, γ) = exp(-(x'² + γ²y'²) / 2σ²) · cos(2πx'/λ)

where:
  λ  = wavelength of the sinusoid    (controls detected frequency)
  θ  = orientation of the filter     (0° to 157.5° in 8 steps)
  σ  = standard deviation of Gaussian envelope
  γ  = spatial aspect ratio          (set to 0.5)
  x' = x·cos(θ) + y·sin(θ)
  y' = -x·sin(θ) + y·cos(θ)
```

By applying 40 such filters (5 frequencies × 8 orientations) and extracting **mean, standard deviation, energy, and entropy** from each response patch, we build a 160-dimensional texture descriptor per patch. Normal fabric produces consistent descriptors; defects break the texture pattern and appear as outliers.

## 📊 Results

| Model | Approach | Notes |
|-------|----------|-------|
| **One-Class SVM** | `rbf` kernel, `nu=0.1`, `gamma=scale` | Fast, no defect labels needed |
| **Dense Autoencoder** | Bottleneck dim=16, threshold at 95th pct | Better sensitivity to subtle defects |

Both models are trained **only on defect-free images** — this is an unsupervised anomaly detection approach, meaning you don't need labelled defect data to train.

```python
# Supported formats: .png, .jpg, .bmp, .tiff
good_files   = sorted(GOOD_DIR.glob('*.*'))
defect_files = sorted(DEFECT_DIR.glob('*.*'))
```
